Leetcode-108-将有序数组转换为二叉搜索树

发表于 2020-07-03 | 分类于 Leetcode

题目

将一个按照升序排列的有序数组，转换为一棵高度平衡二叉搜索树。

本题中，一个高度平衡二叉树是指一个二叉树每个节点 的左右两个子树的高度差的绝对值不超过 1。

二分思路

将数组中最中间的元素作为根结点构建二叉树, 并递归构建左右子树即可.
